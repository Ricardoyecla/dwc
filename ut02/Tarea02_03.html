Tarea02_03

Sumario

  • Prepara un documento PDF para poder ir guardando capturas de todos los pasos que realices en esta tarea.
  • Crear una cuenta Github.
  • Crea una carpeta en tu ordenador llamada parAdwC
  • Instala GitHub en tu ordenador.
  • Inicia la carpeta llamada parAdwC para utilizar con Git y haz la configuración inicial de usuario y email de usuario.
  • Crea otro carpeta dentro de parAdwC con en el nombre UT02
  • Crea dentro de UT02 haz una pagina web donde solo debe aparecer una etiqueta h1 con tu nombre y apellidos.
  • Haz un commit desde parAdwC y sube a GitHub todo lo que hay dentro.
  • Desde la página web de Git crea un fichero de texto dentro de UT02 llamado nuevo.txt y dentro estará tu nombre
  • Haz un pull dentro del directorio parAdwC y modifica la página web y el fichero de texto, añadiendo un parafo con "Desarrollo Web Cliente"
  • Pega en el directoria UT02 el documento pdf con las capturas de realizar los pasos anteriores.
  • Vuelve a hacer un commit y un push para que suban a GitHub

Contenidos 7 puntos uno por apartado

  • Crear cuenta Git
  • Instalar Git en Ordenador
  • Inicia Git en carpeta parAdwC
  • Realiza primer commit correctamente
  • Crea en la web de Git archivo nuevo.txt
  • Realiza pull y trae fichero nuevo.txt
  • Realiza último commit y push

Contenido documento PDF 1 punto 0,5 por apartado

  • Formato del documento, nombre, número de página, margenes, justificación
  • Tamaño imágenes (se captura lo necesario no todo)

  • Dirección de GitHub.
