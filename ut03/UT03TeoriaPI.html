¿Cómo ejecutar código JavaScript?

Vamos a tener dos formas principales de ejecutar código JavaScript la primera es desde el navegador y la otra es utilizando Node.js desde la consola

Debemos indicar que la segunda forma a nosotros desde la parte del cliente que es desde la que vamos a programar es la menos importante, aunque nos puede ser de utilidad para hacer pruebas rápidas.

Ejecutar JavaScript desde el navegador

Desde el navegador con la etiqueta <script>

Podemos colocar la etiqueta <script> en el Head o Body.

En el Head se suele cargar librerías con funciones globales. La carga de este código se hace de forma asincrona por lo que aunque este al final de la página puede que se ejecute antes de que se carge todos los elementos

En el ejemplo escribimos dentro del documento.

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Ejemplo ejecución directa</title>
    </head>
    <body>
        <h1>Página con Javascript</h1>
        <p>JavaScript escribe en la página</p>.</p>
            <script>
                document.write("Yo he escrito");
            </script>
        <p>Página completa.</p>
    </body>
</html>
Mostrar información en la consola

Los navegadores disponen de una Consola que nos van a permitir depurar la ejecución de los programas mostrando información en ella. Pulsamos F12 para mostra la consola o en el menú de opciones del navegador seleccionamos opciones de depuración

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Ejemplo ejecución directa</title>
    </head>
    <body>
        <h1>Página con Javascript</h1>
        <p>JavaScript escribe en la página</p>.</p>
            <script>
               console.log("Aparece en la consola no en el documento");
            </script>
        <p>Página completa.</p>
    </body>
</html>
JavaScript en un fichero externo

Podemos utilizar en la etiqueta <script> el atributo <src> donde podemos indicar la ubicación del fichero con extensión js que queremos cargar.


        <script src="js/fichero.js"><script>
Atributos de <script> : async, defer, integrity y corossorigin.

Ejecutar JavaScript sin utilizar el navegador

Vamos a utilizar Node.js

Tenemos que instalar Node.js

Podemos ejecutar node y se nos abrirá un consola donde poder ejecutar comandos de forma interactiva.

$ node
> console.log("Me llamo Node");
Me llamo Node
undefined

Para salir de la consola pulsamos Control + C

Ejecutar código dentro de un fichero.

Introducimos los comandos en un fichero de texto, es recomendable, que la extensión sea js

$ node nombre_fichero.js
