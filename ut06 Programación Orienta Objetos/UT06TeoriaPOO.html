Creación de Objetos sin constructor.

Se usa para crear objetos pequeños directamente mediante el uso de asignaciones de propiedades y/o métodos a un objeto o cuando se declara el objeto, en los dos casos a medida que se va programando.

Dos formas de crear sin constructor objetos

Ejemplo

Añadir una propiedad.

Para crear una propiedad añadimos al nombre del objeto el operador punto más el nombre de la propiedad, también podemos inicializar.

poligono_regular.lados=4;

poligono_regular.nombre='cuadrado'

Añadir un método.

Para crear un método añadimos el nombre del objeto el operador punto más el nombre del método seguido = function () { instrucciones }

poligono_regular.area=function () {
    return poligono_regular.lados * poligono_regular.lados
}
Ejemplo de creación de un objeto en varias instrucciones.

Ejemplo de creación de un objeto en una sola instrucciones.

Se podría crear todo de una de la siguiente manera.

En este caso dentro de las llaves separados por comas añadimos los nombres de las propiedades y seguido de dos puntos añadimos los posibles valores o funciones que serán los métodos. 

let poligono_regular = {

   	tamanoLado:2,
   	nombre:'cuadrado',

   	}

Uso de This

Es el objeto contexto de JavaScript en el cual se está ejecutando el código actual.

Lo utilizamos para acceder a las propiedades y métodos independientemente de cual sea el nombre del objeto, de esta formas nos abstraemos del nombre del objeto.

 let poligono_regular = {
   	lados:4,
   	tamanoLado:2,
   	nombre:'cuadrado',
   	area:function () {
        return this.tamanoLado * this.tamanoLado
   	}
};

Creación de Objetos con constructor.

Puedes crear un objeto con estos dos pasos:

  • Definir el tipo de objeto escribiendo una función constructora. Existe una fuerte convención, con buena razón, para utilizar en mayúscula la letra inicial.
  • Crear una instancia del objeto con el operador new.

Creación objeto

Creación

Para definir un tipo de objeto, crea una función para el objeto que especifique su nombre, propiedades y métodos.

Ejemplo 
//Función Constructora.

function Poligono_regular(lados,nombre) {
    this.lados=lados;
    this.nombre=nombre;
    this.mostrarInfo=()=>`Soy un ${this.nombre} y tengo ${this.lados} lados\n`;
}

Observa el uso de this para asignar valores a las propiedades del objeto en función de los valores pasados ​​a la función.

El operador instanceof.

Utilice instanceof cuando necesite confirmar el tipo de un objeto en tiempo de ejecución.

let a=new Poligono_regular(4,'Cuadrado');
let b=new Poligono_regular(3,"Triangulo");

console.log(a instanceof Poligono_regular);
console.log(a instanceof String);
¿Qué son los prototipos?.

Es el mecanismo que utiliza javascript para implementar la herencia. El prototipo es la parte común de los objetos del mismo tipo.

Para acceder al prototipo de un objeto utilizamos la propiedad __proto__.

Ejemplo


function Poligono_regular(lados,nombre) {
	this.lados=lados;
    this.nombre=nombre;
    this.mostrarInfo=()=>`Soy un ${this.nombre} y tengo ${this.lados} lados\n`;
}

let a=new Poligono_regular(4,'Cuadrado');
let b=new Poligono_regular(3,"Triangulo");

console.log(a instanceof Poligono_regular);
console.log(a instanceof String);

console.log(a.__proto__);
