Tarea05_02 Con funciones y métodos para arrays

Sumario

Haz una página web para realizar operaciones sobre arrays (matrices de una sola dimensión) de números.

  • El programa pedirá los elementos del array como una cadena de números separados por comas, espacios en blanco, o ... (lo que tu decidas).
  • Solo se podrán hacer dos operaciones: multiplicar una matriz por un escalar z y dividir una matriz por un escalar z cualquiera
  • Ejemplo de multiplicación: 3 x [2, 1, 5 ] = [6, 3, 15]
  • Ejemplo de división: [2, 4, 6] /2 = [ 1, 2, 3]
  • Se deben utilizar funciones siempre que se pueda
  • Para procesar la información de los arrays se utilizara metodos avanzado de manipulación de datos en estrucutras (Tipo filter, foreach, map, ...
