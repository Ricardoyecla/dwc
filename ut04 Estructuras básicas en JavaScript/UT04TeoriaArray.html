Arrays

El objeto Array de JavaScript es un objeto global que es usado en la construcción de arrays, que son objetos tipo lista de alto nivel.

Descripción

Los "arrays" son  objetos tipo-lista,cuyo prototipo tiene métodos para efectuar operaciones de recorrido y de mutación. Ni la longitud de una matriz de JavaScript ni los tipos de sus elementos son fijos. Dado que la longitud de una matriz puede cambiar en cualquier momento, y los datos se pueden almacenar en ubicaciones no contiguas en la matriz, no se garantiza que las matrices de JavaScript sean densas; esto depende de cómo el programador elija usarlos.

Creación y operaciones básicas

Creación

Dos formas de crear Arrays

Syntax

var array_name = [item1, item2, ...];

var array_name = new Array(item1, item2, …)

Ejemplos

var cars = new Array("Saab", "Volvo", "BMW");

var cars = ["Saab", "Volvo", "BMW"];

Acceder a un elemento

var name = cars[0];

Modificar un elemento del Array

cars[0] = "Opel";

Uso de nombres como indice

var person = {firstName:"John", lastName:"Doe", age:46};

Nombre = person[“firstName”]

Bucle sobre un Array
cars.forEach(function (elemento, indice, array) {
    console.log(elemento, indice);
});
Eliminar Elemento de un Array

var cars = ["Saab", "Volvo", "BMW"];

delete cars[2];

Valores indefinidos en un Array

var cars = ["Saab", "Volvo", "BMW"];

delete cars[1];

console.log(cars[1]); // undefined

console.log(cars[2]); // BMW

Un Array puede contener diferentes tipos de elementos.

myArray[0] = Date.now;

myArray[1] = myFunction;

myArray[2] = myCars;

Declarar Array como const.

const cars = ["Saab", "Volvo", "BMW"];

var cars2 = [“hola”,”juan”];

cars[1]=”Abarth”;

cars = cars2 ; //Esto da error
